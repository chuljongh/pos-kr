# 웹사이트 개선 작업 메모

## 🔒 상시 원칙 (모든 작업에 적용)

- **SEO + AI 검색 최적화 최우선**: 초기부터 매우 신경 쓴 영역. 모든 카피·구조·메타·스키마 변경 시 검색(네이버/구글)과 AI 검색(ChatGPT/Perplexity/Gemini) 노출을 먼저 고려.
  - 제목/H1/메타 description에 핵심 키워드 자연스럽게 유지
  - 새 섹션 추가 시 의미적 HTML (article/section/h2-h3 위계), FAQPage·Product·Organization JSON-LD 보강 검토
  - 카피 변경 시 기존 랭킹 키워드 손실 여부 점검
- **용어 통일**: "후정산" 쓰지 말고 **"후불"**로 통일. 사용자에게 익숙한 어휘 우선 ("선불/후불").

## 카피/메시지

- [ ] **비교 표 수정**: "초기 도입 비용", "운영 유지 비용" 항목의 RFID/바코드 시스템 표기를 **"없음" → "낮음"** 으로 변경 (더 정직, 신뢰감 ↑)
  - 위치: `index.html` / `hamba/index.html` 비교 테이블 (Row 1, Row 2)
- [ ] **선불/후불 모두 지원 = 유일한 서비스** 메시지 부각
  - 카피 톤: "**선불과 후불 고객 모두 지원하는 유일한 서비스 · 모바일 QR**" 같은 식으로 익숙한 어휘로
  - 선불(개인/외부 방문객) + 후불(B2B 계약 매장) → 하나의 키오스크로 통합 다이어그램 신설 고려
- [ ] **CCTV 연동 키오스크 언급 추가**: 부가서비스로 CCTV 연동 키오스크 제공 → **기록의 100% 정확성**, 모든 매장 상황에 맞출 수 있는 확장성 어필
